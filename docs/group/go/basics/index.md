## 基本语法
```go
package main

import "fmt"

func main() {
	//  1.这是单行注释

	/*
		2.多行注释
		3.块注释
	*/

	//  4. 在函数内声明变量必须使用，否则无法编译成功
	fmt.Println("Welcome Golang")
}
```
格式化Go语言的格式命令
```go
go fmt main.go
```
