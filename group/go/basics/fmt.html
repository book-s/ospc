fmt #

Go语言的fmt包是一个格式化输入输出库，提供了类似于C语言printf和scanf函数的格式化输出和输入功能。该库主要用于控制台输出和字符串格式化。

fmt包的常用函数有：
1、fmt.Print 用于向控制台输出内容，不会自动添加换行符。
2、fmt.Println 用于向控制台输出内容，会自动添加换行符。
3、fmt.Printf 用于按照指定的格式向控制台输出内容。
4、fmt.Sprintf 用于将格式化的字符串保存到一个字符串变量中。
5、fmt.Sprint 用于将多个参数格式化为一个字符串并返回。
6、fmt.Errorf 用于格式化错误信息，并返回一个error类型的值。

fmt.Printf的格式化占位符
在fmt.Printf中，可以使用占位符来指定输出内容的格式，常用的占位符有：

  • %d 用于输出十进制整数。
  • %f 用于输出浮点数。
  • %s 用于输出字符串。
  • %p 用于输出指针地址。
  • %v 用于输出任意类型的值。
  • %T 用于输出变量的类型。
  • %% 用于输出百分号%。
go
// 下面是一些使用fmt.Printf的示例：
package main

import "fmt"

func main() {
    fmt.Printf("%d\n", 123)   // 输出十进制整数
    fmt.Printf("%.2f\n", 3.14)  // 输出浮点数，保留两位小数
    fmt.Printf("%s\n", "hello world")  // 输出字符串
    fmt.Printf("%p\n", &fmt.Printf)  // 输出指针地址
    fmt.Printf("%v\n", "hello world")  // 输出任意类型的值
    fmt.Printf("%T\n", "hello world")  // 输出变量的类型
    fmt.Printf("100%%\n")  // 输出百分号%
}

字符串定义 #

在Go语言中，字符串是由单个字节连接起来的字符序列，使用UTF-8编码表示Unicode文本。

在Go语言中定义字符串有两种方式：

TIP

1、双引号
双引号中的内容被视为一个字符串，可以包含任何字符，包括Unicode字符。

go
str := "hello world"
1

2、反引号
反引号中的内容被视为一个原始字符串字面量，可以包含多行文本和特殊字符，但不能包含反引号本身。

go
str := `hello world`
1

在Go语言中，字符串是不可变的，也就是说，一旦创建了一个字符串，它的内容就无法改变。如果需要修改字符串，可以将其转换为一个可变的切片或字节数组，进行修改后再转换回字符串。

go
str := "hello"
bytes := []byte(str)
bytes[0] = 'c'
str = string(bytes)
fmt.Println(str) // 输出 "cello"

字符串使用方法 #

Go语言的字符串使用非常方便，以下是一些常用的字符串使用方法：

DANGER

1、获取字符串长度
使用len()函数获取字符串的长度，也就是它包含的字节数。

go
str := "hello"
length := len(str)  // 获取字符串长度
fmt.Println(length) // 输出 5

2、字符串拼接
使用+运算符可以将两个字符串拼接成一个新的字符串。

go
str1 := "hello"
str2 := "world"
str3 := str1 + " " + str2 // 字符串拼接
fmt.Println(str3)        // 输出 "hello world"

3、截取字符串
可以使用切片操作获取字符串的一部分。

go

fmt.Println(substr) // 输出 "hello"

    fmt.Printf("str[%d] = %c\n", i, c) // 输出字符串中每个字符

10、字符串判断
可以使用strings包中的函数来判断一个字符串是否包含另一个字符串、是否以某个前缀或后缀开始或结束等。

go
str := "hello world"
if strings.Contains(str, "world") {
    fmt.Println("str contains world")
}
if strings.HasPrefix(str, "hello") {
    fmt.Println("str starts with hello")
}
if strings.HasSuffix(str, "world") {
    fmt.Println("str ends with world")
}

13、字符串转义
Go语言中的字符串支持一些转义字符，如\n表示换行、\t表示制表符等。如果需要在字符串中包含这些字符本身，可以使用反斜杠进行转义。

go
str := "hello\nworld"
fmt.Println(str) // 输出 "hello" 和 "world" 之间有一个换行符
1
2

14、字符串常量
Go语言中的字符串常量使用双引号包裹，与其他语言类似。同时，也可以使用反引号(`)来定义一个多行字符串常量。

go
str := "hello world" // 使用双引号定义字符串常量
fmt.Println(str)

str2 := `hello
world` // 使用反引号定义多行字符串常量
fmt.Println(str2)
