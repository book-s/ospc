Samba #

Samba是一种用于在Linux和Windows之间共享文件和打印机的软件。

Ubuntu安装Samba #

sh
sudo apt-get update
sudo apt-get install samba
1
2

查看安装的samba版本

sh
samba -V
samba --version
smbd --version
以上三个命令都可以查看samba版本信息
1
2
3
4

创建Samba用户 #

Samba用户必须是ubuntu系统用户，才能添加到samba数据库中生效。

要允许用户访问Samba共享，必须在Samba密码数据库中创建一个用户。可以使用以下命令创建Samba用户：

sh
sudo smbpasswd -a username
其中“username”是要创建的用户名。

2

共享和权限 #

  • 创建一个共享目录
  • 设置共享文件权限
sh
sudo mkdir /home/shared_folder
sudo chmod 777 /home/shared_folder

Samba配置 #

编辑Samba配置文件,使用sudo vim /etc/samba/smb.conf打开配置文件，添加如下内容到配置文件末尾

sh
[global]
   workgroup = WORKGROUP              / 工作组

[share]                               / 共享文件夹
   comment = Shared File              / 共享文件描述
   path = /home/share/sharedfile      / 共享文件路径
   browseable = yes                   / 制定共享文件是否在浏览器中可见
   read only = no                     / 指定是否允许只读访问
   guest ok = no                      / 指定是否允许访客访问共享
   valid users = share                / 指定可以访问共享目录的用户列表

Samba配置多用户访问

sh
[global]                              / 全局设置名称
   workgroup = WORKGROUP              / 工作组名称

[share]                               / 共享文件夹
   comment = Shared File              / 共享文件描述
   path = /home/share/sharedfile      / 共享文件路径
   browseable = yes                   / 制定共享文件是否在浏览器中可见
   writable = yes                     / 共享文件夹设置为可写入状态
   read only = no                     / 指定是否允许只读访问
   guest ok = no                      / 指定是否允许访客访问共享
   valid users = share, share1        / 指定可以访问共享目录的用户列表
   write list = share                 / 读写权限用户
   read list = share1                 / 只读权限用户
   create mask = 0664                 / 创建文件默认权限
   directory mask = 0775              / 创建文件夹默认权限
   force user = share                 / 选项来确保在共享文件夹中创建的所有文件都属于指定的用户和组
   force group = users                / 选项来确保在共享文件夹中创建的所有文件都属于指定的用户和组

设置开机自动挂载磁盘分区 #

执行如下命令，查询磁盘分区的UUID
blkid 磁盘分区

sh
root@servershaer:/home/share/file/datafile# blkid /dev/sdb1
/dev/sdb1: UUID="781b2623-aeca-46f7-bb7a-2ee5d03a08e2" BLOCK_SIZE="4096" TYPE="ext4" PARTLABEL="opt" PARTUUID="a241c0cb-0318-4ec3-bc9b-12d164528391"
1
2

执行以下命令，使用VI编辑器打开“fstab”文件

sh
sudo vim /etc/fstab
1

加入如下代码挂在磁盘

sh
UUID="781b2623-aeca-46f7-bb7a-2ee5d03a08e2" /home/share/file ext4 defaults 0 2
